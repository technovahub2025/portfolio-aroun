# Arunkumar Appadoure - Portfolio Website

A modern, cyberpunk-themed portfolio website showcasing the professional journey and ventures of Arunkumar Appadoure.

## Features

- **Modern Design**: Cyberpunk-inspired UI with neon effects and animations
- **Responsive**: Fully responsive design for all devices
- **Interactive Elements**: Custom cursor, particle effects, and smooth animations
- **Professional Content**: Comprehensive portfolio showcasing business ventures and achievements

## Sections

- **Home**: Hero section with animated typewriter effect
- **Profile**: Professional summary and academic background
- **Company**: Showcase of multiple business ventures
- **Timeline**: Career journey and leadership roles
- **Contact**: Professional contact information

## Technologies Used

- HTML5
- CSS3 with custom animations
- Bootstrap 5
- Font Awesome icons
- AOS (Animate On Scroll)
- Vanilla JavaScript

## Business Ventures Featured

- AROUN SYSTEMS - Industrial Safety & Fire Equipment
- TECHNOVA HUB - IT Solutions & Digital Transformation
- AROUN DISTRIBUTORS - Pan-India Supply Chain
- AROUN PPE - Personal Protective Equipment
- AROUN STORES - Retail Safety Equipment
